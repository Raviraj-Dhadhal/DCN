# CCNA Quiz Practice Website

I am a student, and I am learning computer networks from Cisco.

While preparing for the CCNA exam, I noticed that there was not a simple question bank website where students could practice before the exam. That motivated me to create this project.

This website is made so that everyone learning from Cisco can practice CCNA-style questions and answers from one place. My goal is to help students revise, test themselves, and feel more confident before the exam.

The quiz questions are shuffled each time the page loads, so learners get a mixed practice experience every time they use the website.

## Included Quizzes

- Quiz 1: Introduction to Networks
- Quiz 2: Switching, Routing, and Wireless Essentials
- Quiz 3: Enterprise Networking, Security, and Automation
- Quiz 4: CCNA 200-301 Certification Practice Exam

## Purpose

This project is built for practice, revision, and learning. I created it from my own motivation as a student so other learners can also benefit from an easy and simple quiz website.
